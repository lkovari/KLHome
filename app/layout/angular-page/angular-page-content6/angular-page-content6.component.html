Act #6 Multiple part validation of nested components on a reactive (formerly Model-Driven) Form

We have a main form, which contains a Form. The Form contains a Tab component with three Tabs, each Tab contains one component. Each component on the Tabs contains various data fields. Some data field is mandatory. You must complete all mandatory fields on the Main form and on each Tab for submitting the form data!
