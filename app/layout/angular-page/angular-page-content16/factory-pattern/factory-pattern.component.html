Description
  • Define an interface for creating an object, but let subclasses decide which class to instantiate. Factory Method lets a class defer instantiation to subclasses.
  • Felületet határoz meg egy objektum létrehozásához, de az alosztályokra bízza, melyik osztált példányosítják. A gyártófügvények megengedik az osztályoknak, hogy a példányosítást az alosztályokra ruházzák át.
    • Use Case
  • If the Factory method is implemented, then we don't use a new operator to instantiate the class, instead of we use Factory methods. This pattern is useful when we would instantiate the class, but we don't want to decide which class want to instantiate, because it depends on other parameters for example.
