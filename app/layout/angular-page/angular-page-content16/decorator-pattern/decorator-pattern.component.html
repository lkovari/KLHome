Description
  • Attach an additional responsibilities to an object dynamically. Decorators provide a flexible alternative to subclassing for extending functionality.
  • Az objektumhoz dinamikusan további felelőségi köröket rendel. A kiegészítő szolgáltatások biztosítása terén e módszer rugalmas alternativája az alosztályok létrehozásának.
    • Use Case
  • Decorator pattern is used for extending a legacy functionality without changing the legacy class. So we use decorator pattern, wrap the existing class inside the decorators.
