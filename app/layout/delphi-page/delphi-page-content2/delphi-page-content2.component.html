Debug Event Tracer
Debug Event Tracer, able to detect vanishing effect of Delphi applications on Windows, and can show, where occurred the problem. The vanishing evvect is, when an application simply disappeared, exit without any error notification. In this case a Delphi program in the background find the proper exception on the exception chain, and if not found the proper exception handler, then ammit a last chance exception which this application able to catch, and if the detailed source map is loaded then can show the exact location of the problem.
