Description
  • Compose objects into tree structures to represent part-whole hierarchies. Composite lets clients treat individual objects and compositions of objects uniformly.
  • Az objektumokat faszerkezetbe rendezi, hogy ábrázolhassuk a rész-egész viszonyokat. A módszer révén az önálló objektumokat és az objektumösszetételeket egységesen kezeljük.
    • Use Case
  • If the Factory method is implemented, then we don't use a new operator to instantiate the class, instead of we use Factory methods. This pattern is useful when we would instantiate the class, but we don't want to decide which class want to instantiate, because it depends on other parameters for example.
