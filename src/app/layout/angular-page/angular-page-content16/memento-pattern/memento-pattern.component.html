Description
  • Without violating encapsulation, capture and externalize an object'sinternal state so that the object can be restored to this state later.
  • Az egyégbe zárás (betokozás avagy enkapszuláció) megsértése nélkül kinyeri és rögzíti egy objektum belső állapotát, hogy az később ebbe az állapotba visszaállítható legyen.
    • Use Case
  • The memento design pattern is clearly heavily used in any IDEs, as can be seen when you constantly use CTRL+ Z, to undo mistakes you have made either intentionally or unintentionally. You can imagine that the editor is the caretaker, while the code itself is the memento.
