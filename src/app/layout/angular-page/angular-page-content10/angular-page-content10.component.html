You can see a textarea HTML element. The textarea default height is 3 rows. In this example the textarea decorated with the 'appCustomTextArea' directive. It has two parameters autoSize: boolean and autoSizeLimit: number. With the spinner can change the value of the autoSizeLimit. When the user typing more rows, and the rows number matched with the calue of the autoSizeLimit then stop growing of the textarea height.

Value is required!
